Nested class

Exercise

  1. Create a nested class inside a SCSS file. Such as class header2 is located inside header1.
  2. Change the font color for all the headers inside the body to blue.
  3. Create 4 paragraphs. The first with class "p1" and the second and third with class "p2" and the last with class named "p4".
  4. Give a red color to the second and third paragraph.
